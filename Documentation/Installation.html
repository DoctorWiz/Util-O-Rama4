Installation

Util-O-Rama is distributed as a portable zip file.  In case you don't know what 'portable' means, it means you can extract the zip file to any folder you want, wherever is convenient for you, and run it from there.  Util-O-Rama does NOT include an installer at this time although that is planned for later.

Prerequisites:

Util-O-Rama uses the newest Microsoft .Net version 6.  Because this is fairly new, there is a good chance you do not already have it installed.  If you do, it may need updating.  Go here and download the Microsoft .Net version 6 runtime and install it.

If you wish to use the Vamp-O-Rama or Vamperizer application you will also need to download and install the 64-bit version of the Vamp Plugins (preferably ALL of them, but at a minimum you will need all the Queen Mary ones) which you can get here, and the 64-bit version of Sonic Annotator which you can get here.

Download the Util-O-Rama zip file here.

Using Windows File Explorer Make a new folder named Util-O-Rama somewhere on your computer.  Suggested locations would be in C:\Program Files or C:/PortableApps or in your Christmas directory.  You can even put it right on your desktop if you want (although I don't particularly recommend that).  Once you create the new folder, open it, and leave it open in File Explorer

Open the downloaded zip file and copy the contents to the new folder.  The easiest way is to select all the file in the zip window and drag-drop them to the new folder you have open in File Explorer.

The Util-O-Rama programs are in beta state.  If you don't understand the above directions, you probably have no business trying to run beta software.  Please wait until the software is a bit more mature and has an installer.
