Tune-O-Rama uses Sonic Annotator and the Queen Mary Vamp Plugin to perform the mathmatical analysis of your audio files.
Sonic Annotator and the Queen Mary Vamp Plugin must be installed in order for Tune-O-Rama to work. If you haven't done so already, please download the Queen Mary Vamp Plugin here and install it to C:\Program Files (x86)\Vamp Plugins.
Please download Sonic Annotator here and install it in a convenient location. (Suggested install location is C:\PortableApps\SonicAnnotator)
