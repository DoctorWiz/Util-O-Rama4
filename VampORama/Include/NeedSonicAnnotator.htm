Vamp-O-Rama uses Sonic Annotator and the Queen Mary and other Vamp Plugins to perform the mathmatical analysis of your audio files.
Sonic Annotator and the Queen Mary Vamp Plugins must be installed in order for Vamp-O-Rama to work. If you haven't done so already, please download the 64-bit version of the Vamp Plugin Pack here and install it to C:\Program Files\Vamp Plugins.
Please download the 64-bit version of Sonic Annotator here and install it in a convenient location. (Suggested install location is C:\PortableApps\SonicAnnotator)
For more detailed instructions, please see the Vamp-O-Rama help page.
[Do not download or install the 32-bit versions, they won't work correctly with 64-bit Vamp-O-Rama.]
